# Resilient Technologies

- Asset mapping and GIS - tools for identifying and visualizing community resources

- Community wifi - locally built internet infrastructure

- Mesh networks - decentralized communication networks where devices connect directly to each other. Helpful for when centralized system fails

- Setwardship: the practice of caring for shared resources over time so that ecosystems, infrastructures, and communities can continue to thrive. Share responsiblity over ownership, maintenance.
